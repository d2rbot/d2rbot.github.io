How to update db

plz backup files before update

  1. X:\DB2.xxxx\D2RB\Settings backup Settings folder （Include all bots config files)
  2. X:\DB2.xxxx\day\Settings.json backup it (db bot client control station config files)
  3. Copy old version db ~/day/settings.json and ~/D2RB/Settings/ folder To new version db ~/day/settings.json and ~/D2RB/Settings/ folder

Tips:

  1. We normal named db bot folder as db,rename db folder as Old(db3.6), rename New (db3.7) as db when New(db3.7) is available,
  2. Copy db3.6 ~/day/settings.json and ~/D2RB/Settings/ folder to db ~/day/settings.json and ~/D2RB/Settings/ folder
